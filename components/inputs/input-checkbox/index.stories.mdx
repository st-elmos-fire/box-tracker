# InputCheckbox

<WorksWith component="InputFactory" />

The `InputCheckbox` component is a simple checkbox input with a label.

<Tip type="note">
  The `checked` story is not attached to a changeable status, so it will always
  be rendered as checked. When attached to a status, it will function as a
  toggle.
</Tip>

It is possible to use a Chip as label

<Canvas withSource="open">
  <Story name="With Chip Unchecked">
    {(args) => {
      const [checked, setChecked] = useState(false);
      return (
        <InputCheckbox
          useChip={true}
          label="Unchecked"
          checked={checked}
          onChange={(e) => setChecked(e.target.checked)}
        />
      );
    }}
  </Story>
  <Story name="With Chip Checked">
    {(args) => {
      const [checked, setChecked] = useState(true);
      return (
        <InputCheckbox
          useChip={true}
          label="Checked"
          checked={checked}
          onChange={(e) => setChecked(e.target.checked)}
        />
      );
    }}
  </Story>
  <Story name="With Chip Disabled">
    {(args) => {
      const [checked, setChecked] = useState(false);
      return (
        <InputCheckbox
          useChip={true}
          label="Disabled"
          checked={checked}
          disabled={true}
          onChange={(e) => setChecked(e.target.checked)}
        />
      );
    }}
  </Story>
  <Story name="With Chip coloured">
    {(args) => {
      const [checked, setChecked] = useState(false);
      return (
        <InputCheckbox
          useChip={true}
          label="With colour"
          checked={checked}
          chipColour="#FF0000"
          onChange={(e) => setChecked(e.target.checked)}
        />
      );
    }}
  </Story>
</Canvas>
