# Input Label

The Input Label component is used to label an input. It can also be used to create a 'fake' label where a real
label is not available / appropriate.

<Canvas withSource="open">
  <Story name="Default">{Template.bind({})}</Story>
</Canvas>

You can add a 'required' asterisk to the label by passing the 'required' prop to the component.

<Canvas withSource="open">
  <Story
    name="Required"
    args={{
      required: true
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

The label can also be a larger label (which looks like a form Legend) by passing the 'largeLabel' prop to the component.

<Canvas withSource="open">
  <Story
    name="Large"
    args={{
      largeLabel: true
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

If you need something that looks like a label at a time where a true label isn't appropriate, you can simulate a label
by passing the 'simulated' prop to the component.

<Canvas withSource="open">
  <Story
    name="Simulated"
    args={{
      simulated: true
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>
