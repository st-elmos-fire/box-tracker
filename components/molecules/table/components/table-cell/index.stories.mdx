## Table Cell

The table-cell component is used to display a single cell of a table. It is not a standalone component, but rather a child of the table component.

The TabelCell component can render as a regular (`td`) cell or as a header (`th`) cell.

You can tell the component to render as a header cell by passing the `th` prop.

You don't need to pass the `th` prop if you are rendering the component
inside the `Table` component as it will automatically detect the correct tag and the `TableGenerator`
component will take care of all rendering anyway.

<Canvas withSource="open">
  <Story name="Render as TH">

        <tr>

    </table>

</Canvas>

The `Table` component is designed to be fully responsive, at smaller screens
the table will collapse into a single column. In order for your table cells
to work correctly in this context, you should pass a `contextHeading` prop
to each cell, with the value of its corresponding header cell.

If you are using the `TableGenerator` component (which is strongly recommended),
this will be done automatically for you.
