<Title type="ALPHA">Grid</Title>

<Description of={Grid} />

The Grid component will automatically adjust the number of columns to fit the screen size, the column value specified by the user
is the number of columns displayed on an average desktop screen or above.

<Tip type="warning">
  In these examples, the children expand to fit the entire grid, however this
  will depend on the child components themselves.
</Tip>

<Tip type="note">
  The grid displays a grey background colour in the stories. However this is just as a visual aid for the developer. The grid
  component does not actually render any background colour unless the user specifies it.
</Tip>
